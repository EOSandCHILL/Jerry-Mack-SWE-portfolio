# Jerry Mack Software Engineering Portfolio

Personal developer portfolio for Jerry Mack, a Marine Corps Veteran and
Software Engineer supporting DOJ USTP systems.

The site is a static HTML/CSS/JavaScript portfolio that highlights federal
software engineering, Java modernization, secure SDLC work, cloud-connected
delivery, and a responsible AI engineering practice through personal AI-enabled
software projects.

## Current Positioning

- Federal software engineering and regulated government delivery
- Java 8/11/17, Spring, JSP, Struts, DAO/service-layer maintenance
- Vulnerability remediation with Sonatype IQ, SonarQube, dependency upgrades,
  and secure code quality improvements
- Bamboo, Azure Government / DOJ environments, WildFly, ActiveMQ, Application
  Insights, Gradle, and Maven workflows
- NSF EPSCoR / EDOCS experience with Spring Boot, PostgreSQL, jOOQ, Liquibase,
  Vue 3, TypeScript, AWS, Jenkins, Docker, and automated testing
- AI engineering practice focused on AI APIs, Azure AI Fundamentals / AI-900,
  RAG, automation, and secure AI-enabled government use cases

## Project Structure

- `index.html` - main portfolio content and page structure
- `css/stylesheet.css` - primary site styling
- `css/ss.css` - Superslides slideshow styling
- `js/script.js` - loader, slideshow initialization, typed hero labels, and
  smooth scrolling
- `js/typed.js` - local Typed.js dependency
- `js/ss.js` - local Superslides dependency
- `images/` - project images, icons, resume assets, and visual media
